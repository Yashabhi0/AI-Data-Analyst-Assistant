# AI Data Analyst Assistant

An AI-powered data analysis tool that automatically analyzes datasets, generates visualizations, detects patterns, identifies anomalies, and produces human-readable insights and reports.

The system combines automated exploratory data analysis (EDA), intelligent visualization, anomaly detection, and an interactive dashboard to help users quickly understand their data.

---

## Features

### Automated Data Analysis
- Dataset profiling (rows, columns, data types)
- Missing value detection
- Duplicate detection
- Statistical summaries

### Visualizations
Automatically generates multiple charts:

- Histogram
- Box Plot
- Bar Chart
- Scatter Plot
- Correlation Heatmap
- Pair Plot
- Violin Plot
- Missing Values Heatmap
- Boxplot by Category

### Interactive Dashboard
Built using **Streamlit** with features like:

- Dataset upload (CSV / Excel)
- Kaggle dataset loading
- Dataset preview
- KPI summary cards
- Organized visualizations
- Interactive charts
- Downloadable analysis reports


### Intelligent Features
- Automatic anomaly detection
- Anomaly explanations
- Natural language dataset queries
- Automatic chart recommendations

---

## System Architecture

The system uses a **multi-agent architecture** where each agent performs a specific task in the analysis pipeline.
